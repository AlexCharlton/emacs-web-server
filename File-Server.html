4.4 File Server

The following example implements a file server which will serve files from the docroot document root set to the current working directory in this example. Three helper functions are used; ws-subdirectoryp is used to check if the requested path is within the document root, if so the file is served and ws-send-file is used to appropriately set the mime-type of the response based on the extension of the file, if not then ws-send-404 is used to send a default “File Not Found” response. 

;;; file-server.el --- serve any files using Emacs Web Server
(lexical-let ((docroot default-directory))
  (ws-start
   (list (cons (cons :GET ".*")
               (lambda (request)
                 (with-slots (process headers) request
                   (let ((path (substring (cdr (assoc :GET headers)) 1)))
                     (if (ws-subdirectoryp docroot path)
                         (ws-send-file process (expand-file-name path docroot))
                       (ws-send-404 process)))))))
   9004))
